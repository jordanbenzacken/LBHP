## LOCAL
Possibilité de lancer en local :
* `git clone
* `npm install` (la première fois)
* `npm start`

* `npm test` pour lancer les tests
* si FSEVENTS error, installer watchman :
https://facebook.github.io/watchman/docs/install.html
